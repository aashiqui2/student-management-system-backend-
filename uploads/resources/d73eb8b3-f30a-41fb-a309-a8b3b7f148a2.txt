# SSH Key Setup for Git + GitHub:

Step 1 : Create a new folder and open Git Bash/CMD and goto the folder location

Step 2 : Initialize Git  
         git init

Step 3 : Add some files in the folder

Step 4 : Run commands 
         git status
         git add .
         git commit -m “…..” 

Step 6 : Add the remote repo URL 
         git remote add origin <link>

Step 7: Generate SSH Key:(Recommended: Ed25519 algorithm)
        ssh-keygen -t ed25519 -C "your_email@example.com"
        click enter twice store keys(public and private) in home directory with default name and no extra password(passpharse):
        path:[/c/Users/Aashiq/.ssh/id_ed25519.pub]

Step 8: paste the ssh public key in github account

Step 9: Test SSH Connection
        ssh -T git@github.com

Step 10:then Push your code to remote repo
        git push origin master

# SSH Key Setup for Git + GitHub using custom name for keys and passpharse:

Step 1 : Create a new folder and open Git Bash/CMD and goto the folder location

Step 2 : Initialize Git  
         git init

Step 3 : Add some files in the folder

Step 4 : Run commands 
         git status
         git add .
         git commit -m “…..” 

Step 6 : Add the remote repo URL 
         git remote add origin <link>

Step 7: Generate SSH Key:(Recommended: Ed25519 algorithm)
        ssh-keygen -t ed25519 -C "your_email@example.com"
        Enter the entire path:  C:\Users\Aashiq\.ssh\<new_keyname>
        Enter the passpharse: <extra password>

Step 8: paste the ssh public key in github account

Step 9: Test SSH Connection
        ssh -i ~/.ssh/<key_name> -T git@github.com

Step 10: Configure SSH (Recommended for multiple keys)
         nano ~/.ssh/config
         [CTRL+O To write] click Enter to paste the command
``
Host github.com
    HostName github.com
    User git
    IdentityFile ~/.ssh/<new_name_file>
    IdentitiesOnly yes
```
         
         and [CTRL+X to exit] and [type Y to save]

Step 11:Test SSH Connection
        ssh  -T git@github.com

Step 10:then Push your code to remote repo
        git push origin master
        Enter the passpharse to push 


If you dont want to create a config and type the passpharse again and again then Go to ssh agent:

Step 10:Start SSH Agent 
        eval "$(ssh-agent -s)"

Step 11: Add SSH Key to Agent
        ssh-add ~/.ssh/<key_name>
        and type passpharase(password) only once

Step 12: Check loaded keys:
         ssh-add -l

Step 13:then Push your code to remote repo
        git push origin master

Step 14:To stop ssh agent 
       ssh-agent -k
